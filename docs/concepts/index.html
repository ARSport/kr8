    Concepts

    kr8 has two main concepts you should be aware of before you get started:

    • components
    • clusters

    The relationship between components and clusters are simple: components are installed on clusters. You will have components that are install on all clusters, and some components will only be installed on some clusters.

    Components can be declared multiple times within a cluster, as long as they are named distinctly.

    Clusters are unique and singular
